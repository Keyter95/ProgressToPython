# ProgressToPython

Progress ABL to Python Converter
As a Progress ABL developer, it has become crucial for me to be able to showcase my skillset in other languages relevant to the industry.

The thinking behind this project was to be able to show I understand the syntax differences between Progress and Python, and that I am able to manually convert from Progress ABL into working Python code.
The project features a UI screen that offers a developer various options to convert their code. A developer can paste the code they want translated into the provided area, or upload a Progress file. Further the output can be displayed on the screen, downloaded as a .py file or copied to a clipboard to be used later.

The frontend is React.js based, integrated with JS functions and the backend is Python based which gave me the ability to learn how to use an external API.

Currently in Beta, I am busy working on a translator that translates code back from Python to Progress. Making this project a two way translator rather than just a one way converter.
